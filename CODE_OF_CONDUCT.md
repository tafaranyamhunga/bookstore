# Code of Conduct

## My Pledge

This repository is a personal project shared publicly.
Everyone is expected to interact respectfully and professionally.

## Expected Behavior

- Be respectful and considerate
- Avoid harassment or offensive language
- Keep discussions constructive and on-topic

## Unacceptable Behavior

- Harassment or discrimination of any kind
- Trolling or intentionally disruptive behavior
- Sharing illegal or harmful content

## Enforcement

I reserve the right to remove comments, issues, or contributions that violate this code of conduct.
