# Security Policy

## Reporting Concerns

If you notice:
- Sensitive or private content accidentally committed
- Copyright concerns
- Any other security-related issue

Please **do not open a public issue**.

Instead, contact me directly so the issue can be handled quickly and privately: [Email](mailto:tafaranyamhunga@proton.me)

## Scope

This repository:
- Does not contain application code
- Does not process user data
- Does not accept uploads from users

As such, no formal vulnerability disclosure process is required.
