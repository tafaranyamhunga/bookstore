# Contributing

Thanks for your interest in this repository!

This is primarily a **personal digital bookstore**, so contributions are kept simple and structured.

## Before Contributing

Please follow these steps before making any changes:

1. **Check the category** where the book belongs and confirm that it does not already exist.
2. Ensure the book is provided **only in PDF format**.
3. Make sure the book fits the purpose of this repository.

## Adding a Book

### If the category already exists:
- Add the book PDF to the appropriate category folder
- Create a Pull Request with a clear description

### If the category does NOT exist:
- Create a new branch
- Follow the existing directory structure
- Add the new category folder
- Place the book PDF inside it
- Open a Pull Request explaining the new category

## What You Can Do

- Suggest book recommendations via Issues
- Add books to existing categories
- Propose new categories (following the steps above)
- Report missing, broken, or incorrect files
- Raise copyright concerns

## What You Should Not Do

- Upload copyrighted material without permission
- Add non-PDF files
- Restructure directories without discussion
- Submit large or unrelated changes

## Final Notes

All contributions are reviewed and approved at the discretion of the repository owner.
This repository is meant to stay clean, organized, and easy to navigate.

Thank you for contributing
